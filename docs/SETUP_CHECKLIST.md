# Setup Checklist

## Supabase

- [ ] Conta Supabase criada
- [ ] Projeto Supabase criado
- [ ] SUPABASE_URL obtida
- [ ] SUPABASE_ANON_KEY obtida
- [ ] SUPABASE_SERVICE_ROLE_KEY obtida
- [ ] schema.sql executado no SQL Editor

## Conta Azul

- [ ] Conta Azul Pro ativa
- [ ] Acesso ao Portal do Desenvolvedor
- [ ] App criado no Portal da Conta Azul
- [ ] CLIENT_ID obtido
- [ ] CLIENT_SECRET obtido
- [ ] REDIRECT_URI local configurado: `http://localhost:3000/api/conta-azul/callback`
- [ ] REDIRECT_URI de produção configurado depois

## Vercel

- [ ] Conta Vercel criada
- [ ] GitHub conectado
- [ ] Projeto conectado na Vercel
- [ ] Variáveis de ambiente configuradas na Vercel

## Ambiente local

- [ ] Node.js instalado
- [ ] Dependências instaladas (`npm install`)
- [ ] `.env.local` criado e preenchido
- [ ] App rodando localmente (`npm run dev`)
- [ ] `/integrations` abrindo no browser
- [ ] Botão "Conectar Conta Azul" funcionando
- [ ] OAuth completo — token salvo no Supabase
- [ ] Status mostrando "Conectado"
