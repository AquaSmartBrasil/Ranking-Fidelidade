# Conta Azul Dashboard

## Objetivo

Criar um app privado para conectar com a API da Conta Azul, puxar dados de vendas e financeiro, salvar no Supabase e depois exibir um dashboard atualizado.

## Stack

- Next.js
- TypeScript
- App Router
- API Routes no Next.js
- Supabase Postgres
- Supabase Auth
- Vercel
- Conta Azul API com OAuth 2.0
